# SpendSense

> AI Agent that stops impulse spending before it happens.

Built for the Google Cloud Agent Builder Hackathon 2026.
Partner track: Fivetran.

## What it does
SpendSense is an AI agent that monitors your spending across
all Indian e-commerce and quick commerce apps, detects impulse
spending patterns using Gemini 3.5 Flash, and fires a smart
nudge before you make a purchase you'll regret.

## Tech Stack
- Agent: Google Antigravity 2.0
- AI: Gemini 3.5 Flash
- Data pipeline: Fivetran MCP
- Database: MongoDB Atlas
- Backend: Python + FastAPI
- Frontend: React PWA
- Hosting: Google Cloud Run
